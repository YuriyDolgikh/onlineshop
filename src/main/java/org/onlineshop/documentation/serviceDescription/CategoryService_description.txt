Описание сервиса CategoryService

Сервис CategoryService отвечает за управление категориями товаров в системе.
Он реализует интерфейс CategoryServiceInterface и обеспечивает операции создания,
обновления, удаления и получения категорий. Также сервис взаимодействует с 
репозиторием и вспомогательным классом CategoryServiceHelper для обработки изображений.

Основные зависимости:

* CategoryRepository — для доступа к данным категорий.
* CategoryConverter — для преобразования сущностей в DTO и обратно.
* CategoryServiceHelper — для обработки изображений категорий (метод resolveImageUrl).

Основные методы:

1. addCategory(CategoryRequestDto categoryRequestDto)
   Добавляет новую категорию.
   * Проверяет корректность входных данных.
   * Проверяет, не существует ли уже категории с таким именем.
   * Обрабатывает изображение категории через helper.resolveImageUrl().
   * Создаёт новую категорию и сохраняет её в базе данных.
   * Возвращает DTO созданной категории.

2. updateCategory(Integer categoryId, CategoryUpdateDto categoryUpdateDto)
   Обновляет существующую категорию по её идентификатору.
   * Проверяет, существует ли категория с указанным ID.
   * Проверяет корректность нового имени (длина от 3 до 20 символов, уникальность).
   * Обновляет имя и/или изображение категории, если они переданы.
   * Сохраняет изменения и возвращает DTO обновлённой категории.

3. deleteCategory(Integer categoryId)
   Удаляет категорию по её идентификатору.
   * Проверяет, что ID не равен null.
   * Проверяет, существует ли категория в базе данных.
   * Переносит все продукты из удаляемой категории в категорию “Other”.
   * Удаляет категорию из базы данных.
   * Возвращает DTO удалённой категории.

4. getAllCategories()
   Возвращает список всех категорий.
   * Извлекает все категории из базы данных через CategoryRepository.
   * Преобразует их в DTO и возвращает в виде списка.

5. getCategoryById(Integer categoryId)
   Возвращает категорию по её идентификатору.
   * Выполняет поиск категории через CategoryRepository.
   * Если категория не найдена, выбрасывает исключение BadRequestException.

6. getCategoryByName(String categoryName)
   Возвращает категорию по её имени.
   * Выполняет поиск категории в базе данных.
   * Если категория не найдена, выбрасывает исключение BadRequestException.
