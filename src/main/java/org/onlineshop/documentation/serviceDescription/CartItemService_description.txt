Описание сервиса CartItemService

Сервис CartItemService отвечает за управление товарами в корзине пользователя. 
Он реализует интерфейс CartItemServiceInterface и использует несколько зависимостей
 для работы с корзинами, товарами и пользователями.

Основные зависимости:

* CartItemRepository — для сохранения и удаления элементов корзины.
* UserService — для получения текущего пользователя.
* CartItemConverter — для преобразования сущностей в DTO.
* CartService — для работы с корзиной пользователя.
* ProductService — для проверки существования и получения данных о товаре.

Основные методы:

1. addItemToCart(CartItemRequestDto cartItemRequestDto)
   Добавляет товар в корзину текущего пользователя.
   * Проверяет корректность входных данных.
   * Проверяет, существует ли товар по переданному productId.
   * Если товар уже есть в корзине, увеличивает его количество.
   * Если товара нет — создаёт новый элемент корзины и сохраняет его.
   * После этого обновляет корзину и возвращает DTO добавленного товара.

2. removeItemFromCart(Integer productId)
   Удаляет товар из корзины по идентификатору товара.
   * Проверяет, что ID не равен null.
   * Находит корзину текущего пользователя и удаляет из неё нужный элемент.
   * Удаляет соответствующий CartItem из базы данных.
   * Обновляет корзину и возвращает DTO удалённого товара.

3. updateItemInCart(CartItemUpdateDto cartItemUpdateDto)
   Обновляет количество выбранного товара в корзине.
   * Проверяет корректность входных данных.
   * Проверяет, существует ли товар и находится ли он в корзине.
   * Обновляет количество и сохраняет изменения.
   * Возвращает обновлённый CartItem в виде DTO.

4. getCartItems()
   Возвращает список всех товаров в корзине текущего пользователя в виде множества DTO (Set<CartItemResponseDto>).

5. getCartItemFromCart(Integer productId) [приватный метод]
   Вспомогательный метод для получения конкретного элемента корзины по ID товара из корзины текущего пользователя.
