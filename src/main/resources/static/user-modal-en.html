REST Endpoints

POST /v1/users/register PUBLIC

Register a new user. User receives email with confirmation code.

POST /v1/users/login PUBLIC

User authentication and JWT token retrieval.

GET /v1/users/confirmation?code={code} PUBLIC

Email address confirmation using confirmation code.

PUT /v1/users/{userId} USER

Update user information. Users can only update their own profile.

DELETE /v1/users/{userId} USER

Delete user account. ADMIN can delete any user.

GET /v1/users/renew/{email} USER

Restore deleted user account.
